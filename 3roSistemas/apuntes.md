### Tipos de Datos
En JavaScript hay diverso tipos de datos. Entre los cuales encontramos:

1. Entero (Integer)
2. Decimal (coma flotante - Float)
3. Cadena (Texto - String)
4. Objeto (object)
5. Arreglo (Array)
6. Boolean
7. NULL

### Operadores

En JavaScript existen operadores aritmeticos, los cuales son:

1. Adicion (+)
2. Sustraccion (-)
3. Multiplicacion (*)
4. Divicion (/)
